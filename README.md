This project is a responsive Temperature Converter Website developed as part of Task 3 of the Oasis Infobyte Web Development Internship.
The application allows users to easily convert temperatures between Celsius, Fahrenheit, and Kelvin through a simple and intuitive interface.

Built using HTML, CSS, and JavaScript, the project focuses on clean UI design, smooth user interaction, and accurate real-time conversions. Special attention was given to usability, minimal styling, and clarity of output.
